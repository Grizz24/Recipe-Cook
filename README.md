# Recipe-Cook

## 🚀 Overview
Welcome to Recipe Cook, a CLI application that lets you store recipes and keep track of ingredient inventory.

## 🛠 Tech Stack
- C#
- MySQL

## ✨ Features
- To enter and store recipes
- To display all the information about the recipes
- To adjust the Calories and Quantity of each recipe


To use the app, follow these instructions

1. To enter the recipes
a.Enter the name of the recipe
b.Number of quantity (It only allows a single value)
c.Number of Calories (It only allows a single value)
d.Enter the food groups of the recipes
e.Enter the steps of the recipes
f.Click save to store all the information
e.If you want to enter another recipe, follow the steps

2. To Display the Recipes
a.After saving the recipes you entered, click the display button to display
The recipe names in the Recipe Textbox

3. To Display all the information of the Recipes
a.In the Recipe textbox, choose the recipe you want to display
b. The recipe you chose will display in the Display textbox
and the steps of the recipes will be displayed in the display steps

4. To adjust the Calories and Quantity
a.Click the adjust button, which will take you to a new page
b.On the new page, choose what number you want to adjust the Calories and Quantity by choosing the ratio page
c.Click the adjust button to adjust the values
d.Click the show button to show the values
e.You can choose which recipe values to adjust
f.To exit the application, click the exit button
